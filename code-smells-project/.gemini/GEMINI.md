# GEMINI Context: code-smells-project

## Project Overview
This is a Python/Flask-based E-commerce API designed as a legacy codebase for refactoring challenges. It focuses on demonstration of "code smells" and architectural anti-patterns.

## Technical Stack
- **Language:** Python
- **Framework:** Flask 3.1.1
- **Database:** SQLite (loja.db)
- **Features:** Flask-CORS enabled.

## Setup & Execution
1. Install dependencies: `pip install -r requirements.txt`
2. Run application: `python app.py`
3. Port: `http://localhost:5000`

## Key Information
- The SQLite database is automatically created and seeded on the first boot.
- The project is part of the `refactor-arch` challenge.
