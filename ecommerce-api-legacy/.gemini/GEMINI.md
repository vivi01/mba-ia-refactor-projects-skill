# GEMINI Context: ecommerce-api-legacy

## Project Overview
This is a Node.js/Express LMS (Learning Management System) API with a checkout flow. It serves as a legacy codebase for refactoring exercises.

## Technical Stack
- **Language:** JavaScript (Node.js)
- **Framework:** Express 4.18.2
- **Database:** SQLite3 (In-memory)
- **Project Type:** Legacy Boilerplate

## Setup & Execution
1. Install dependencies: `npm install`
2. Run application: `npm start`
3. Port: `http://localhost:3000`

## Key Information
- Uses an in-memory SQLite database that seeds automatically on startup.
- Request examples can be found in `api.http`.
- Part of the `refactor-arch` challenge.
